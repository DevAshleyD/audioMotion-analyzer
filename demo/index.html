Note: demo scripts are bundled with webpack. The uncompressed source code is located in the /demo/src folder.

▶ Fluid layout demo

  • Illustrates usage of audioMotion-analyzer in a fluid layout context - resize the window and see how the analyzer adjusts itself;
  • Load an audio file from your computer, or a remote file or stream via URL;
  • Try out all visualization and configuration options in real time;
  • Play test tones generated by a frequency oscillator node additionally connected to the analyzer;
  • See how to use the onCanvasDraw callback to draw additional content on the canvas (the pulsating audioMotion logo).

▶ Multiple instances demo

  • Demonstrates three analyzer instances sharing the same audio context, but each with its own configuration settings.

▶ Overlay demo

  • Uses the overlay property to display the analyzer over a video element;
  • Defines a custom gradient, with semi-transparent background color;
  • Manages full-screen mode.
